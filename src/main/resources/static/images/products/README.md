# Thư mục ảnh Sản phẩm

Đặt các ảnh sản phẩm vào đây.

## Quy tắc đặt tên:
- `product-{id}.jpg` - Ví dụ: product-1.jpg, product-2.jpg
- Hoặc tên mô tả: `banh-tet.jpg`, `nem-chua.jpg`

## Kích thước đề xuất:
- Chiều rộng: 800px
- Chiều cao: 800px
- Tỉ lệ: 1:1 (vuông)
- Format: JPG, PNG, hoặc WEBP

## Lưu ý:
- Ảnh nên có nền trắng hoặc trong suốt
- Sản phẩm nên chiếm 70-80% diện tích ảnh
- Tối ưu kích thước file (dưới 200KB)
