## Lưu ý:

- Tất cả logo nên có nền trong suốt (transparent background)
- Kích thước file nên dưới 50KB để tối ưu tốc độ tải
- Nếu logo không tồn tại, hệ thống sẽ tự động ẩn và chỉ hiển thị text
- Logo sẽ được tự động scale để phù hợp với giao diện

## Cách thêm logo:

1. Tải logo từ các nguồn trên hoặc từ website chính thức của đơn vị vận chuyển
2. Resize logo về kích thước phù hợp (120x40px recommended)
3. Lưu file với tên chính xác như trên vào thư mục này
4. Refresh trang admin để thấy logo mới

## Fallback:

Nếu không có logo, hệ thống sẽ chỉ hiển thị tên đơn vị vận chuyển dưới dạng text.
