# Thư mục ảnh Slider

Đặt các ảnh slider vào đây với tên file:

- `slider-1.jpg` - Slide đầu tiên
- `slider-2.jpg` - Slide thứ hai  
- `slider-3.jpg` - Slide thứ ba

## Kích thước đề xuất:
- Chiều rộng: 1920px
- Chiều cao: 600-800px
- Tỉ lệ: 16:9 hoặc 21:9
- Format: JPG, PNG, hoặc WEBP

## Lưu ý:
- Ảnh nên có độ phân giải cao để hiển thị đẹp trên màn hình lớn
- Nên tối ưu kích thước file (dưới 500KB) để tải nhanh
- Nội dung chính nên ở giữa ảnh để không bị che bởi text overlay
