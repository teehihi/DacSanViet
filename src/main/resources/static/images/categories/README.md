# Thư mục ảnh Danh mục

Đặt các ảnh danh mục sản phẩm vào đây.

## Quy tắc đặt tên:
- `category-{id}.jpg` - Ví dụ: category-1.jpg, category-2.jpg
- Hoặc tên mô tả: `mien-bac.jpg`, `mien-trung.jpg`, `mien-nam.jpg`

## Kích thước đề xuất:
- Chiều rộng: 600px
- Chiều cao: 400px
- Tỉ lệ: 3:2
- Format: JPG, PNG, hoặc WEBP

## Lưu ý:
- Ảnh nên thể hiện đặc trưng của danh mục
- Tối ưu kích thước file (dưới 150KB)
